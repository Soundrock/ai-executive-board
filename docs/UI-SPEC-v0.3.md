# 智策中心 UI Spec v0.3

## UI 設計方向

主畫面採用簡單、集中、任務優先的設計。

不要做成複雜 Dashboard。
不要一開始顯示大量功能模組。
首頁應該像 ChatGPT 一樣，讓使用者專注在單一輸入口。

## 主畫面核心

中間為大型輸入區。

支援：
- 文字輸入
- 貼上內容
- 拖曳圖片
- 拖曳 PDF
- 拖曳 Word
- 拖曳 Excel
- 拖曳 PPT
- 郵件內容
- 語音輸入

系統自動判斷任務類型。

## 右側控制面板

包含：

主控 AI：
- ChatGPT
- Gemini
- DeepSeek
- Claude
- 豆包
- Grok

參與 AI：
- 可複選

研究等級：
- 快速模式
- 標準模式，預設
- 深度研究

輸出模式：
- 主控整合
- 顯示全部答案
- 比較表
- AI 評分
- 辯論模式
- 投票模式

進階設定需收合，不放在首頁干擾使用者。

## 設計原則

所有事情，一個入口。

使用者不用先選文件、圖片、Prompt 或程式模式。
AI 自動判斷。

主畫面要簡單、清楚、低干擾。
