# Contributing

Thank you for your interest in contributing to AI Executive Board.

This project is in early active development. Contributions, feedback, bug reports, documentation improvements, and feature ideas are welcome.

## Contribution Areas

- Documentation
- UI improvements
- Multi-agent workflow improvements
- AI routing logic
- Testing
- Security review
- Open-source setup improvements

## Development Guidelines

- Do not commit API keys, tokens, credentials, or private data
- Keep local .env files private
- Use .env.example for environment variable examples
- Keep changes focused and easy to review
- Explain major changes clearly in pull requests

## Project Goal

The goal is to build an open-source AI consensus platform that allows multiple AI systems to work together and help users make better decisions.
